모두의 마음가짐(Moeum) 개인정보 처리방침

시행일: 2026-05-18

1. 수집·이용하는 개인정보

모두의 마음가짐은 회원가입을 요구하지 않으며, 사용자의 개인정보를 수집하지 않습니다. 사용자가 앱 내에 입력·저장하는 모든 텍스트와 이미지는 사용자의 iOS 기기 내부 저장소에만 보관됩니다.

2. 카메라 및 사진 라이브러리 접근

  • 카메라: 사용자가 직접 ‘사진 찍기’를 선택했을 때만 일시적으로 카메라를 사용하며, 촬영된 이미지는 글자 추출(OCR)에만 쓰이고 기기 내에서만 처리됩니다.
  • 사진 라이브러리: 사용자가 선택한 사진의 글자만 추출하며, 라이브러리 전체를 스캔하지 않습니다.
  • 추출된 텍스트와 이미지 메타데이터는 외부 서버로 전송되지 않습니다.

3. 알림

사용자가 ‘매일 알림’을 활성화한 경우 iOS 로컬 알림 시스템을 사용합니다. 알림은 사용자 기기 내부에서 예약·발송되며, 외부 푸시 서버를 사용하지 않습니다.

4. 외부 서버·제3자 공유

모두의 마음가짐은 어떤 외부 서버에도 사용자 데이터를 전송하지 않으며, 제3자에게 제공·판매·임대하지 않습니다. 광고 SDK·분석 SDK·크래시 리포팅 SDK를 포함하지 않습니다.

5. 데이터의 보관·삭제

  • 모든 데이터는 사용자의 iOS 기기에 보관됩니다.
  • 앱 삭제 시 데이터는 iOS 시스템에 의해 함께 삭제됩니다.
  • 사용자가 앱 내 ‘설정 → 내 데이터’에서 JSON으로 내보내거나 개별 문장을 직접 삭제할 수 있습니다.

6. 아동의 개인정보

모두의 마음가짐은 만 14세 미만의 사용자로부터도 별도의 개인정보를 수집하지 않습니다. 단, 본 앱은 일반 사용자를 대상으로 합니다.

## 4. 설명 (Description, 4000자 제한)

```
좋은 문장은 한순간에 만나서, 다시 찾을 때면 어디서 봤는지 기억나지 않아요.

모두의 마음가짐은 책·신문·SNS·기사에서 발견한 문장을 사진 한 장으로 저장하고, 매일 다시 만나게 해주는 단순한 한국어 앱입니다.

■ 핵심 기능

· 사진에서 자동 추출 (OCR)
  책 페이지, 스크린샷, 자막을 카메라로 찍거나 사진첩에서 고르면 글자를 자동으로 추출합니다. 페이지 번호 제거·공백 정리·줄바꿈 다듬기로 깔끔하게 정돈할 수 있어요.

· 매일 알림
  설정한 시간에 저장해 둔 문장 한 줄을 알려드려요. 잊고 있던 좋은 글귀를 우연처럼 다시 만나는 순간을 만듭니다.

· 듣기 (TTS)
  한국어 음성으로 문장을 읽어줍니다. 산책 중에도, 운전 중에도 모아둔 문장을 들으세요. 폴더 전체 연속 재생도 지원합니다.

· 폴더로 정리
  주제별로 폴더를 만들어 문장을 분류하세요. ‘오늘의 영감’, ‘책 메모’, ‘좋은 카피’처럼 자유롭게.

· 공유·내보내기
  한 문장은 카카오톡·메일·메시지로 곧바로 공유. 전체 컬렉션 또는 폴더별로 JSON 파일을 내보내거나 친구와 주고받을 수 있어요(중복은 자동 건너뜀).

· 다크모드 · Pretendard 폰트
  한국어 가독성에 최적화된 Pretendard 폰트와 라이트/다크/시스템 테마.

■ 개인정보 약속

· 모든 문장과 사진 처리는 기기 안에서만 이루어집니다.
· 외부 서버로 데이터를 전송하지 않습니다.
· 회원가입·로그인 없음. 광고 없음. 추적 없음.

■ 이런 분께

· 책에서 좋은 구절을 발견해도 스크린샷만 찍고 마는 분
· 시·에세이·격언을 모으는 습관이 있는 분
· SNS 캡션, 기사 한 줄, 강연 메모를 흩어두기 싫은 분
· 매일 좋은 문장 한 줄로 하루를 시작·마무리하고 싶은 분

■ 시리즈

모두의 임장 · 모두의 가계부 · 모두의 여행 · 모두의 마음가짐 — 일상을 모으는 4부작 중 ‘문장’ 편.

좋은 문장을 모으다.
```

## 부록 A — 개인정보 처리방침 본문 초안

> `privacy.html` 또는 PRIVACY.md로 배포. 도메인은 GitHub Pages 권장.

```
모두의 마음가짐(Moeum) 개인정보 처리방침
시행일: 2026-05-18

1. 수집·이용하는 개인정보
모두의 마음가짐은 회원가입을 요구하지 않으며, 사용자의 개인정보를 수집하지 않습니다.
사용자가 앱 내에 입력·저장하는 모든 텍스트와 이미지는 사용자의 iOS 기기 내부 저장소에만 보관됩니다.

2. 카메라 및 사진 라이브러리 접근
- 카메라: 사용자가 직접 ‘사진 찍기’를 선택했을 때만 일시적으로 카메라를 사용하며, 촬영된 이미지는 글자 추출(OCR)에만 쓰이고 기기 내에서만 처리됩니다.
- 사진 라이브러리: 사용자가 선택한 사진의 글자만 추출하며, 라이브러리 전체를 스캔하지 않습니다.
- 추출된 텍스트와 이미지 메타데이터는 외부 서버로 전송되지 않습니다.

3. 알림
사용자가 ‘매일 알림’을 활성화한 경우 iOS 로컬 알림 시스템을 사용합니다. 알림은 사용자 기기 내부에서 예약·발송되며, 외부 푸시 서버를 사용하지 않습니다.

4. 외부 서버·제3자 공유
모두의 마음가짐은 어떤 외부 서버에도 사용자 데이터를 전송하지 않으며, 제3자에게 제공·판매·임대하지 않습니다. 광고 SDK·분석 SDK·크래시 리포팅 SDK를 포함하지 않습니다.

5. 데이터의 보관·삭제
- 모든 데이터는 사용자의 iOS 기기에 보관됩니다.
- 앱 삭제 시 데이터는 iOS 시스템에 의해 함께 삭제됩니다.
- 사용자가 앱 내 ‘설정 → 내 데이터’에서 JSON으로 내보내거나 개별 문장을 직접 삭제할 수 있습니다.

6. 아동의 개인정보
모두의 마음가짐은 만 14세 미만의 사용자로부터도 별도의 개인정보를 수집하지 않습니다. 단, 본 앱은 일반 사용자를 대상으로 합니다.

7. 처리방침의 변경
본 처리방침이 변경되는 경우, 변경 사항은 본 페이지를 통해 공지합니다.

GitHub: https://github.com/haenara-shin/moeum
```
